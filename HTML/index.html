They say writing is easy, you sit in front of your paper, in my case a keyboard and a screen, and bleed.


I agree.

To write is to feel, to write is think, to write is to bleed. If only I had all the words, then I could truly say what I think and how I feel. This website is an attempt to put ink...no, blood to paper, as an imperfect human with imperfect words, to describe perfectly the ideas, thoughts, and feelings of my soul. That is quite an endeavor. If you would like to join me, read on. Maybe together, we can learn to speak and write more beauitfully, more clearly, and more truly. The hope is with greater understanding, we can love ourselves and others more deeply.
